
Este proyecto automatiza la evaluación de solicitudes de crédito utilizando 6 modelos de Machine Learning colaborativos alojados en GitHub para la toma de decisiones gerenciales.

## Ejemplo de Aplicación Práctica (Gestión de Riesgos):
* **Cliente Evaluado:** Juan Pérez
* **Ingresos Mensuales:** $4,500 | **Historial:** Bueno | **Deudas:** $1,000

### Resultados de la Clasificación Simultánea en el Sistema:
1. **Regresión Logística (Binaria):** Clasificado inicialmente como **APTO**.
2. **Árbol de Decisión (Perfil):** Clasificado como **Riesgo Bajo** debido a que su nivel de endeudamiento es menor al 50%.
3. **Random Forest (Decisión Final):** **APROBADO** con un monto máximo sugerido de crédito de $45,000.
4. **KNN (Similitud):** Asignado automáticamente al segmento de *Clientes Premium / Pagadores*.
5. **SVM (Hiperplano de separación):** Clasificado *Sin necesidad de Aval* (dentro del margen seguro de riesgo corporativo).
6. **Naive Bayes (Probabilidad de Urgencia):** Marcado como *Prioridad Estándar* de atención en plataforma.
